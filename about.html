ABOUT

브랜드와 기능이 따로 놀지 않게

귤귤은 소개용 랜딩과 운영 기능을 분리하지 않고, 같은 화면 결 안에서 자연스럽게 이어지도록 구성한 시안입니다.

BRAND DIRECTION

랜딩은 브랜드 무드, 중간은 모집, 끝은 실제 참여

프로그램과 참여 흐름 중심으로 구조를 재편했습니다. 프로젝트성 행사, 팝업, 커뮤니티성 모집에 더 어울리는 문맥을 목표로 했습니다.

WHY THIS STRUCTURE

정적이지만 기능은 살아 있는 허브

GitHub Pages에서 바로 올려 검수할 수 있고, 같은 UI 위에 Supabase를 붙이면 프로그램 구성·폼 제출·내 일정 확인까지 서버 기반으로 이어집니다.

PLACEHOLDER

이미지 자리는 전부 비워둠

나중에 실제 비주얼을 넣기 쉽게, 텍스트 플레이스홀더 대신 빈 프레임만 두었습니다. 지금은 구조와 동선만 보는 데 집중할 수 있습니다.

빈 프레임은 전부 동일한 스타일로 정리해 두었고, 실제 사진·배너·포스터로 쉽게 교체할 수 있습니다.
