왜 이번 버전은
UI를 먼저 볼 수 있게 바꿨나

처음부터 키를 넣고 권한을 맞추고 RLS까지 건드리면 초보자는 화면을 보기 전에 지친다. 그래서 이번 패키지는 아무 설정 없이도 데모 모드로 화면과 기능을 전부 확인할 수 있게 바꿨다. 그 뒤에 Supabase를 붙이면 된다.

  • web / admin 둘 다 실제 UI가 바로 열린다
  • 폼 생성, 질문 편집, 신청 제출까지 데모 모드에서 연습 가능
  • Supabase 연결 후에도 폴더 구조는 그대로 유지된다
